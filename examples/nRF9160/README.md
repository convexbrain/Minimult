# nRF9160

## Device

* nRF9160 by Nordic Semiconductor
  * https://www.nordicsemi.com/Products/Low-power-cellular-IoT/nRF9160
  * ARM Cortex-M33 with FPU
    * 64 MHz
    * 1 MByte FLASH
    * 256 kByte RAM
  * nRF MDK
    * https://www.nordicsemi.com/Software-and-tools/Development-Tools/nRF-MDK
