# STM32F746NG

## Device

* STM32F746NG by STMicroelectronics
  * https://www.st.com/ja/microcontrollers-microprocessors/stm32l053r8.html
  * ARM Cortex-M7
    * up to 216 MHz
    * 1 MByte FLASH
    * 320 kByte RAM
  * STM32F7 System View Description
    * https://www.st.com/resource/en/svd/stm32f7_svd.zip
